# ToolBox Pro - 开发者必备工具箱

🚀 一个轻量实用的开发者工具集合，让你的日常开发效率提升一倍！

## 📖 这是什么？

ToolBox Pro 是一个专为网页开发者设计的 Chrome 扩展工具箱，采用模块化设计，持续添加实用开发工具。目前内置两个高频使用的开发工具：

### ✨ 当前功能

**1. Cookie 获取工具**
- 一键获取当前页面的所有 Cookie
- 自动复制到剪贴板，直接粘贴使用
- 支持 Netscape 标准格式输出（用于 curl 等工具）
- 零点击操作，瞬间完成

**2. XPath Helper - 元素选择器生成器**
- 点击页面任意元素，自动生成唯一 XPath
- 同时生成符合 Chrome DevTools 规则的 CSS Selector
- 支持 ID 优先策略，生成更稳定的选择器
- 鼠标悬停高亮预览，一键复制
- 调试爬虫、自动化测试必备

## 🎯 为什么你需要安装它？

作为开发者，你是否经常：

- ❌ 需要手动复制 Cookie，一个个手动拼接格式？
- ❌ 手写 XPath，反复调试才能唯一匹配元素？
- ❌ 在多个扩展之间切换，每个工具只做一件事？
- ❌ 使用笨重的开发者工具面板，操作繁琐？

**ToolBox Pro 解决了这些痛点：**

✅ **开箱即用** - 无需配置，安装即可用
✅ **极速响应** - 轻量代码，瞬间完成操作
✅ **一键复制** - 结果直接到剪贴板，粘贴即用
✅ **干净界面** - 简洁设计，不打扰你的工作流
✅ **持续更新** - 模块化架构，未来会添加更多实用工具

## 💡 使用场景

- 爬虫开发快速复制 Cookie 和元素选择器
- 自动化测试（Selenium/Puppeteer）定位元素
- 接口调试时快速获取认证 Cookie
- 网页分析快速提取元素路径
- 前端开发调试快速验证选择器

## 🔒 权限说明

- `activeTab` - 仅访问你当前打开的活动标签页
- `cookies` - 仅用于读取当前页面 Cookie
- `scripting` - 仅在你主动使用 XPath 工具时注入脚本
- 我们不会收集任何数据，你的所有操作都在本地完成

# ToolBox Pro - Developer's Essential Toolkit

🚀 A lightweight and practical collection of developer tools that doubles your daily development efficiency!

## 📖 What is this?

ToolBox Pro is a Chrome extension toolkit designed specifically for web developers. It features a modular design and continuously adds practical development tools. Currently includes two frequently used tools:

### ✨ Current Features

**1. Cookie Getter**
- One-click to get all Cookies from the current page
- Automatically copied to clipboard, ready to paste
- Supports Netscape standard format output (for curl, etc.)
- Zero-click operation, done in an instant

**2. XPath Helper - Element Selector Generator**
- Click any element on the page to automatically generate a unique XPath
- Simultaneously generates CSS Selector following Chrome DevTools rules
- Supports ID-first strategy for more stable selectors
- Hover highlighting preview, one-click copy
- Essential for debugging crawlers and automated testing

## 🎯 Why should you install it?

As a developer, do you often:

- ❌ Manually copy cookies and splice formats one by one?
- ❌ Write XPath by hand, repeatedly debug to match elements uniquely?
- ❌ Switch between multiple extensions, each tool only does one thing?
- ❌ Use bulky devtool panels with cumbersome operations?

**ToolBox Pro solves these pain points:**

✅ **Ready to Go** - No configuration required, use immediately after installation
✅ **Lightning Fast** - Lightweight code, completes operations instantly
✅ **One-Click Copy** - Results go directly to clipboard, ready to paste
✅ **Clean Interface** - Simple design, won't disturb your workflow
✅ **Continuous Updates** - Modular architecture, more tools will be added in the future

## 💡 Use Cases

- Quickly copy cookies and element selectors for crawler development
- Locate elements for automated testing (Selenium/Puppeteer)
- Quickly get authentication cookies for API debugging
- Quickly extract element paths for web analysis
- Quickly verify selectors for frontend development debugging

## 🔒 Permissions

- `activeTab` - Only access your currently open active tab
- `cookies` - Only used to read current page cookies
- `scripting` - Only inject scripts when you actively use XPath tool
- We don't collect any data, all operations are done locally
