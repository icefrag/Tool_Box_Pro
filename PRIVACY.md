# Privacy Policy / 隐私权政策

## 中文版本

### 数据收集说明

ToolBox Pro 是一个开源的 Chrome 扩展工具，**不会收集、存储、传输或共享任何用户的个人数据或浏览信息**。

### 数据处理说明

本扩展的所有功能都在用户的本地浏览器中完成：

- **Cookie 获取工具**：Cookie 数据仅在本地读取并复制到剪贴板，不会上传到任何服务器
- **XPath Helper**：元素选择操作完全在用户当前标签页本地完成，不会发送任何数据到外部服务器
- **用户设置**：用户的偏好设置仅保存在用户浏览器的 `chrome.storage.sync` 中，仅用于同步用户配置，不会被我们访问

### 第三方服务

本扩展不包含第三方分析、广告或追踪代码。

### 联系我们

如果你对本隐私权政策有任何疑问，请通过 GitHub 项目提交 Issue。

## English Version

### Data Collection Statement

ToolBox Pro is an open-source Chrome extension. **We do not collect, store, transmit, or share any of your personal data or browsing information.**

### Data Processing Description

All functions of this extension are performed locally in your browser:

- **Cookie Getter**: Cookie data is only read locally and copied to your clipboard, it is never uploaded to any server
- **XPath Helper**: Element selection operations are performed entirely locally in your current tab, no data is sent to external servers
- **User Settings**: User preferences are only stored in `chrome.storage.sync` of your browser for syncing your configuration across devices, and we cannot access this data

### Third-party Services

This extension does not contain any third-party analytics, advertising, or tracking code.

### Contact Us

If you have any questions about this privacy policy, please submit an Issue via the GitHub project.
